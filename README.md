Dev Sorteio
Projeto simples de sorteio de números aleatórios, desenvolvido com HTML, CSS e JavaScript, com foco no treino de lógica de programação, manipulação do DOM e tratamento de dados do usuário.

📌 Sobre o projeto
O Dev Sorteio permite que o usuário informe um valor mínimo e um valor máximo, e o sistema retorna um número inteiro aleatório dentro desse intervalo.
O projeto também valida se os valores foram informados corretamente, evitando erros comuns como valores invertidos.

⚙️ Funcionalidades

Definir valor mínimo e máximo
Gerar número aleatório inteiro
Validação de valores (mínimo não pode ser maior que o máximo)
Interface simples e intuitiva
Execução direta no navegador (sem dependências)


🛠️ Tecnologias utilizadas

HTML5 – Estrutura da página
CSS3 – Estilização da interface
JavaScript (Vanilla) – Lógica do sorteio e interação com o DOM


🧠 Conceitos praticados

Manipulação do DOM (querySelector)
Eventos (onclick)
Funções em JavaScript
Math.random() e Math.floor()
Conversão de valores (Number())
Validação de dados do usuário
